Mis proyectos

Algunos de los proyectos que realicé, entre simples y complejos.

Aplicación SIG

La aplicación permite el ingreso y búsqueda de datos con formularios, el trazado de la ruta más corta de un punto a otro, inclusión de imágenes de los puntos espaciales y la publicación web del mapa utilizando QGIS.

Sitio web para Adultos Mayores

Está orientado a la educación tecnológica para personas de la tercera edad por medio de videos, es una plataforma amigable y sencilla de manejar. Herramientas utilizadas: JavaScript, HTML5, CSS3, Bootstrap

Sistema Experto

Sencillo sistema experto de predicción del clima para salir a jugar al aire libre, dependiendo de algunas condiciones ambientales. Herramientas utilizadas: JavaScript, HTML5, CSS3, Bootstrap

Sistema Académico

Sistema académico con funcionalidades diferentes para cada tipo de Rol: Administrador, Alumno, Docente, Director, Decano, Egresado, Registro Central y de facultad, etc. Trabajo realizado en equipo utilizando el framework Zend de PHP en integración con la base de datos PostgreSQL

Sistema de Control de Asistencia

El empleado puede registrar su horario de entrada o salida y presentar una justificación por falta. El administrador puede gestionar los horarios, empleados, puestos,aceptar o rechazar justificaciones. y generar reportes. Herramientas utilizadas: PHP, JavaScript, HTML5, CSS3, Bootstrap, MySQL.

Sistema de Voto Electrónico

Este sistema permite la elección de la junta directiva de alguna organización. El administrador puede gestionar los puestos, candidatos, y ver resultados. Herramientas utilizadas: PHP, JavaScript, HTML5, CSS3, Bootstrap, MySQL.

Sistema Firma digital

Aplicación web implementada a partir del modelo del proceso "Firmas de actas" de una institución de educación superior. Realizado en equipo, con Bizagi Modeler, Bizagi Studio y SQLServer para Rediseñar y automatizar este proceso.

Sistema de Pedidos

Sistema web de pedidos para un Resto-Bar. Manejo de roles de cocina, mozo y administrador. Actualización en tiempo real de estados según la entrega de pedidos y roles. Herramientas utilizadas: PHP, JavaScript, HTML5, CSS3, Bootstrap, AJAX, MySQL.

App Android Pomodoro

Consiste en 25 minutos de trabajo o "pomodoro" y a continuación un tiempo de descanso de 5 minutos, luego del cuarto "pomodoro", el descanso es de 10 minutos. Para esta aplicación usé la clase de Android CountDownTimer, MediaPlayer y la librería RingProgressBar.

App Secretaria

Esta aplicación facilita la redacción de documentos desde cualquier dispositivo móvil a través de plantillas de documentos administrativos para su edición en tiempo real. Realizado en equipo para un Instituto.
